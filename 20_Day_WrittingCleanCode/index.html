Guía de estilo de JavaScript

blue Una guía de estilo de JavaScript es un conjunto de estándars que dicen como debería estar organizado y escrito el código de JavaScript. En esta sección, hablaremos sobre las guías de JavaScript y cómo escribir código limpio.

JavaScript es un lenguaje de programación y como el lenguaje humano tiene una sintaxis. La sintaxis de JavaScript tiene que ser escrita siguiendo una cierta guía de estilos en aras de convinción y simplicidad.

¿Por qué necesitamos una guía de estilos?

Has estado escribiendo código solo por mucho tiempo pero ahora parece que trabajas en un equipo. No importa cómo escribas tu código siempre que funcione, de todas maneras cuando trabajas en un equipo con 10 o 20 desarrolladores en un proyecto y en el mismo código base, el código fuente puede ser un caos y difícil de manejar si no hay ninguna guía a seguir. Puedes desarrollar tu propia guía y convenciones o puedes adaptar buenas guías de desarrollo. Las guías de estilo de JavaScript más conocidas son:

  • Airbnb JavaScript Style Guide
    • Airbnb tiene una de las más populares guías de estilo en Internet. Cubre de cerca cada aspecto de JavaScript muy bien y es adoptado por muchos desarrolladores y compañías. Es recomendable probarlo. Su estilo es fácil de usar y simple de entender.
  • JavaScript Standard Style Guide
    • No tan popular pero funciona muy bien. Retira el punto y coma en su guía de estilos
  • Google JavaScript Style Guide
